## 使用技术
- Vue
- webpack
- leancloud

## 安装部署

1. 下载

git clone git@github.com:NathanYangcn/todoList-vue.git

2. 进入项目

cd todoList-vue

3. 安装依赖

npm install

4. 运行

npm run dev

## 目标功能

- 添加任务
- 删除任务
- 标记完成
- 修改任务标题和详情
- 保存任务
- 筛选任务
- 添加任务清单
- 任务归属（初始添加）
- 登录
- 注册
- 登出
